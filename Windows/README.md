# 二、Windows 网站介绍
首先我了解优秀软件的途径主要是RSS里订阅的关于各种软件介绍的播客，推荐以下几个我比较喜欢的:
### 善用佳软 
简评:虽然博客更新不是很频繁，很多软件都很老，但是质量相当高。介绍非常详细有用。还出了一本书《善用佳软：高效能人士的软件应用之道》写的也是非常的棒。而且在B站的视频也是让人受益匪浅。

### 電腦玩物 
简评:虽然主要介绍手机APP，但是偶尔也会介绍web和电脑软件。对软件的介绍也是非常详细。而且除了介绍软件本身还会介绍一些实际的使用方法，比如时间管理、笔记术之类的。而且还出了实体书《比別人快一步的 Google 工作術：從職場到人生的100個聰明改造提案》(不过这本我还没买……)

### 小众软件 
简评:比较简洁明了的介绍一下软件的功能。各平台的软件都有介绍，更新频率还挺高的，介绍软件的质量也不错。

### 反斗软件 
简评:特点和小众软件差不多，也很不错。

### 异次元软件世界 
简评:特点和上面两个差不多，不过介绍的更加详细，我也挺喜欢的。

### Windows 绝赞应用 
简评:收集了很多优秀的Windows应用，并且很好的分类了。目前仍然在不断更新中，非常推荐。

然后介绍下载的地方……其实感觉没有什么好介绍的，基本上就是在官网下就好了，UWP应用在win商店。linux更多的是直接用包管理器下载安装升级。

好的，接下来进入正题。

## 下载工具
### Persepolis Download Manager 
简评:PDM是一款封装了 Aria2 作为内核，并为其套上图形界面的开源免费下载软件。它能让你享受 Aria2 一切的特性，同时又帮助你完全跳过安装和配置 Aria2 那些繁琐的过程，并且有一个图形化的直观界面供你用鼠标进行操作，你就像用迅雷、Folx 等下载工具一样的简单明了，而不必再对着命令行发愁。
而且关键是PDM为跨平台的软件，更换平台后无需切换浏览器扩展。

## 效率

### VirtualBox 
简评:VirtualBox 完全免费开源，可跨平台运行在 Win、Mac、Linux 系统上。加上该虚拟机功能强大且性能出色，可以流畅虚拟运行各大主流操作系统，如微软全系列 (包括 Windows 10) / Linux 各种发行版 (Ubuntu / CentOS / Deepin ) 甚至是 Android 等，能应对各种场景需求。因此它一直是无数用户首选的虚拟机软件……

异次元软件世界:virtualbox.html

### ditto 
简评:Ditto 是一款开源、免费、强大的剪贴板增强工具。可以把复制过的所有内容保存起来（可以设定保存日期或条目总数），快捷地供后续调用。还可以合并粘贴，纯文本粘贴，支持分组、置顶、快速搜索、热键粘贴功能。并且，还可以通过网络共享剪贴板内容。

善用佳软:ditto.htm
