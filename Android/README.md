## 零、前言  

### 工具

- CalcNote Pro

打开速度和功能性都很不错的计算器软件，多行计算非常好用。可以快速调用之前的计算结果。  
项目链接：<https://play.google.com/store/apps/details?id=com.burton999.notecal.pro>

对二次元人群较为友好的曲库，但是客户端越更新越垃圾。推荐使用 4.0.0 版本。

- PowerAMP

强大的本地播放器
