- 片假名终结者 - 在网页中的日语外来语上方标注英文原词

日语的片假名是非常神奇的东西。日语中有很多的英语外来词，然后日语就会用罗马音的发音来读英语。
于是在看一些片假名单词时需要先翻译成英文，然后再翻译到中文。非常头疼。

这个脚本可以在片假名上标注对应的英文小字。方便阅读。

项目链接：<https://greasyfork.org/zh-CN/scripts/33268-katakana-terminator>

## 网页介绍  

### 搜索引擎  

- AnywhereAnything  

目前暂未找到替代品的主页，输入关键词，然后点网站，即在那个网站里搜索。  
项目链接：<http://lackar.com/aa/>  

- F Search (F搜) - 一款新的搜索引擎

支持中文与英文搜索, 过滤广告和内容农场，支持高级搜索。目前的搜索体验还算不错。让我回忆起了多吉搜索的美好回忆。  
并且这个项目正在初创的时期，给开发者反馈的问题都会很积极地接受。对我这种批话多的人来说可太快乐了，我是非常喜欢尝试使用这种软件的，有种追更的感觉。  
长期支持方面据作者说似乎可以保证后续不会关站。之后如果涉及到盈利的话会考虑增加不影响体验的广告或者出一些付费的增值服务。

除了搜索的本体服务之外还提供了一些代码分享、GitHub加速下载之类的小功能。

项目链接：<https://fsou.cc/>

- Bing  

微软出品的通用搜索引擎。搜索结果质量略弱于谷歌和百度。但无需翻墙、没有广告。(最近有点BUG，替换成上面的搜索引擎了)  
项目链接：<https://cn.bing.com/>  

- Magi  

Magi 是由 Peak Labs 研发的基于机器学习的信息抽取和检索系统。本质上并非搜索引擎，所以搜索结果弱于大部分正规搜索引擎。(本质是玩具)  
项目链接：<https://magi.com/>  

- CC search  

搜索 CC 协议发布的内容专用的引擎。  
项目链接：<https://search.creativecommons.org/>  

- 鸠摩搜书  

搜索网络书籍的专用引擎。  
项目链接：<https://www.jiumodiary.com/>
