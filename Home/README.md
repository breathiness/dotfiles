## 食品

- 牛油果 - 我不推荐的网红水果

近几年牛油果还挺火的，网传作为水果挺健康的。但是作为一个吃货，我对健康其实并不太感兴趣，仅是好奇味道如何。于是在最近购买了几个打算尝尝鲜。在初次尝试之后在我这边是可以拉到难吃一档了。试了几种网上介绍的吃法感觉都不太行。口感方面肉质偏软微微带点脆，算是平平无奇。但是在味道方面没有甜、酸、咸之类的味道。但是那种牛油果特有的风味又很浓。打个不太恰当的比方，有点吃生韭菜的意思。作为辅料给其他食物增添风味可能还行。单独吃是有点难吃了。还有在网上还看到有说法，说是牛油果含有油脂和蛋白质，吃起来会有奶油的味道。我只能说是纯纯的扯淡。就算是有也是被牛油果本身的风味给盖住，尝不出来的。
所以对于这个网红食物。我的评价是“别买”，就算买也建议只买一个试试味道。感觉以后这类在宣传上强调健康，对味道没什么描述的食物我可能都会远离了。

## IT配件

- Logi Craft 键盘 - 我很喜欢，但并不推荐购买系列

前几天我入手了这款有点特别的键盘——带有一个旋钮的 Logi Craft 键盘。
之前看稚辉君的客制化键盘视频，确实非常心动。但是看到发车的价格……只能说客制化是这样的。不过看到这个视频也是让我想起我几年前关注过的一个产品，然后我找了找二手的价格感觉还能接受。于是就果断下手了。
至于为什么我要在这款键盘2017年发布5年后才入手呢？只能说原价 ¥1,699.00 属实让人望而却步。而现在99新400的价格对于一款非机械键盘来说其实也并不能称得上什么好价格。但对我来说还能接受。
然后是功能方面，罗技全家桶很方便，修改配置简单，距离感应的功能也很不错。虽然在买之前我觉得滚轮就是个鸡肋跟快捷键比好像并没有什么明显的优势，但到手后不得不说多了一个滚轮确实还蛮舒服的，跟用快捷键是完全不同的两种操作感。而滚轮的分段和自由滚轮两种不同手感，对有些功能来说是更符合操作直觉的。不过切换时会发出清脆的一声“唰”，刚开始用时偶尔会被吓到。轻触切换功能也很不错。但是按压并旋转的功能就有些鸡肋了。操作起来非常别扭。
然后是手感方面。有很多人都说机械键盘手感好。但是我试用过很多轴体，还是更喜欢薄膜键盘的手感。而 Craft 的手感据说是薄膜键盘的天花板。实际上手试了之后感觉还行，比起普通薄膜要轻和脆，还是不错的。目前最大的问题是重。铝合金的全尺寸键盘重量确实不容小觑。还有一个问题是键帽据说挺容易打油的，我还是有点纠结要不要带个保护套再用。最后对我个人来说缺憾的一点是旋钮右侧一整条的空位还是有点浪费空间的。如果加个类似 Touch bar 的功能那就很不错。也不会像原版 Touch bar 那样导致功能建缺失。

项目链接：<https://www.logitech.com.cn/zh-cn/products/keyboards/craft.html>

- ​LMN-3  - 开源版「OP-1」项目 口袋音乐制造机

最近在midifan看到的一个项目，类似非常出名的 OP-1 便携式多媒体合成器。但是众所周知，OP-1的淘宝代购价格已经到了9500，可以说是非常离谱了。而这个项目就是很好的DIY替代。

“有些人可能会将 LMN 3 视为 Teenage Engineering OP-1 的 DIY 替代品，因为它被设计为用于音乐制作的便携式多合一解决方案。

LMN-3 DAW 基于开源音频引擎 Tracktion Engine，提供录制、编辑和混音等标准功能。LMN-3 DAW  基于 Raspberry PI 硬件平台，只需要树莓派即可打造属于自己的「OP-1」。项目组不仅提供了固件系统和软件，还有帮助制作外观的各类 CAD 文件。动手能力强的朋友可以试试看。 ”——midifan

项目链接：<https://github.com/FundamentalFrequency>
